Bubble Sort (*버블 정렬)
- 특정 배열안에 앞에있는 값과 뒤에있는 값을 연이어서 비교해서 최종적으로 배열 내 값을 정렬하는 것

Select Sort (*선택 정렬)
- 특정 배열안에서 순차적으로 값을 선택해서 배열의 모든 값들과 비교 후 최종적으로 배열 내 값을 정렬하는 것
